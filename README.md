## Description
This application uses the latest technologies in order to provide the back-end for an internet based e-commerce retail company. This back end will allow a manager at such a company to compete with other e-commerce companies. 
## Installation
This back-end application is seeded with existing data that puts items for sale into categories. Each category has its own ID. Each product also has its own IDand belongs to one of the listed categories. Products also have tags, and tag IDs. This application allows the user to view all or specific categories, products, and/or tags. This application also allows the user to create new categores, products or tags, modify existing categores, products, or tags, and delete existing categories, products, or tags. 
## Usage
To run this program, the user will need to install and use MySQL2, Sequelize, Node.js, dotenv.
